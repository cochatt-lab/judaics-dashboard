Judaic Curriculum Unit Designer

Quick Start
1. Choose Text Type (Chumash, Navi, Talmud, Halacha) and Grade.
2. Select Book → Chapter → Start/End Verse (for Halacha: Content / Allocation → Key Source / Unit → Key Concept / Theme).[file:2]
3. Pick one or more Benchmarks that match your unit focus.[file:1]
4. Fill in the unit text boxes (Aim, Outline, Outcomes, etc.).
5. Click Save Unit to download your unit file and upload it to the shared folder.
